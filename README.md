

2. **Install Dependencies**
```bash

```


3. **Environment Configuration**
Create a `.env` file in the root directory and add the following keys:
```env

MONGO_URI=mongodb://localhost:27017/restaurant-bot
SESSION_SECRET=your_random_secret_string
PAYSTACK_SECRET_KEY=sk_test_xxxx...
PAYSTACK_PUBLIC_KEY=pk_test_xxxx...

```


4. **Run the Application**
* **Development Mode (Hot Reloading):**
```bash
npm run dev

```


* **Production Build:**
```bash
npm run build
npm start

```




5. **Access the Bot**
Open your browser and navigate to: `http://localhost:3000`

## ğŸ§ª Testing the Bot

### Navigation Flow

1. **Start:** The bot welcomes you with options.
2. **View Menu:** Type `1`.
3. **Add Item:** Type the ID of an item (e.g., `10` or `11`).
4. **View Cart:** Type `97`.
5. **Checkout:** Type `99` to generate a payment link.

### Payment Simulation

1. Click the **"Pay Now"** link generated by the bot.
2. On the Paystack checkout page, choose **"Success"** to simulate a successful transaction.
3. The application will verify the transaction and redirect you back to the chat with a success message.
